λμηιτ

I've started working on a new xUnit test framework that uses the new lambda syntax in Java 8. The inspiration comes from the Javascript test framework Jasmine, so this is what it looks like:

        describe("A suite", (d) -> {
            it("contains spec with an expectation", (o) -> {
                expect(true).toBe(true);
            });
        });

At present it kind of works, but it's still an early experiment.
